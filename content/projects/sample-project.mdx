# Sample SaaS Dashboard

This project partnered with a seed-stage SaaS team to level up their analytics dashboard experience.

## Highlights

- Reworked onboarding wizard based on user interviews and analytics data.
- Converted slow legacy pages to Next.js server components with caching.
- Added regression test suite powered by Playwright and component coverage.

## Results

- Activation up 18% within the first month post-launch.
- Average dashboard load time dropped from 3.8s to 1.6s.
- Fewer support tickets thanks to clearer reporting presets and empty states.

## Responsibilities

- Product strategy alongside the PM and cofounder.
- UX/UI redesign, prototyping, and stakeholder reviews.
- Development, QA, and launch playbook.

## Tech Highlights

- Next.js 15 App Router
- Tailwind CSS design system components
- Zod for request validation and type-safe forms
- Playwright for end-to-end regression coverage
