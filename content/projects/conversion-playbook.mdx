# Conversion Playbook Revamp

A fast-paced engagement to relaunch a marketing site with better copy, performance, and analytics.

## Deliverables

- Brand-new hero, services, and testimonial sections designed to mirror the sales call flow.
- Global component library including announcement bar, comparison tables, and content blocks.
- Newsletter capture and contact flows hooked into Resend + Plausible with event tracking.

## Outcomes

- +24% increase in demo requests within the first two weeks.
- 98 Lighthouse performance score across core pages.
- Stakeholder onboarding deck & Loom walkthroughs for future iterations.

## Role

Own strategy, UX/UI, and implementation while collaborating asynchronously with the founder and one contractor copywriter.

## Tech Stack

- Next.js 15 static generation
- Tailwind CSS + shadcn/ui
- Resend transactional emails
- Plausible analytics events
