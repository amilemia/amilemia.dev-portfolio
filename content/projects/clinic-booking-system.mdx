# Clinic Management System

A bilingual, high-performance web application designed to streamline clinic workflows, schedule appointments, generate medical prescriptions, and manage billing.

## Key Features

### 1. Interactive Scheduler
The core of the system is a calendar powered by FullCalendar, allowing clinic assistants and doctors to view, create, drag, and resize appointments in real-time with instant AJAX updates.

![Interactive Calendar Month View](/images/projects/month_calendar.png)

![Interactive Calendar Week View](/images/projects/week_calendar.png)

### 2. Dashboard Analytics
A central workspace providing statistical widgets on daily appointments, registered patients, pending invoice counts, and quick navigation shortcuts.

![Clinic Dashboard](/images/projects/dashboard.png)

### 3. Patient & Prescription (Ordonnance) Management
A secure registry of patients and clinical records. Doctors can write digital prescriptions (ordonnances) with automated formatting and download clean PDF versions for print.

![Appointments List](/images/projects/appointments.png)

### 4. Doctor Leave Scheduler
Doctors can register time-offs and vacations, which immediately blocks those slots in the scheduler to prevent overlapping bookings.

---

## Tech Stack & Standards

- **Backend Framework**: Laravel 13 & PHP 8.3
- **Frontend & Styling**: Tailwind CSS, Alpine.js, SweetAlert2
- **Calendar Engine**: FullCalendar (v6) with AJAX Event Handling
- **Reporting**: Barryvdh Laravel DomPDF (PDF Invoices & Receipts)
- **Quality Assurance**: Automated integration tests utilizing Pest/PHPUnit
