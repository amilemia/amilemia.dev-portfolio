# Clinic Management System

This project was built to solve critical administrative bottlenecks and scheduling conflicts for modern medical practices.

## Key Features

- **Multi-Role Dashboards:** Customized views and access controls for Administrators, Doctors, and Receptionist/Assistants.

![Appointments List](/images/projects/appointments.png)

- **Zero-Overlap Calendar:** Real-time visual calendar with conflict-checking validation for appointments and availability.

![Calendar Week View](/images/projects/week_calendar.png)

![Calendar Month View](/images/projects/month_calendar.png)

- **Prescription & Billing:** Secure generation of patient prescriptions and associated invoice payments.
- **Bilingual Interface:** Instant locale switching (English/French) for all navigation and form components.

## Results

- Administrative time spent on scheduling dropped by over 90%.
- Patient appointment conflicts reduced to zero.
- 100% passing rate in automated security vulnerability tests.

## Tech Stack & Quality Standards

- Built with modern PHP 8.x, MySQL, and semantic HTML5/CSS3.
- Secure session tokens, CSRF protection, and SQL injection prevention.
- Custom-made testing suite verifying system installation, database health, security protocols, and login controls.
