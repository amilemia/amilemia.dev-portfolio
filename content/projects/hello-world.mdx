# Hello World Project

This is a sample project demonstrating how to use Contentlayer2 with Next.js 15. The project showcases various features including:

- MDX support with frontmatter
- Type-safe content
- Automatic type generation

## Features

- **Modern Stack**: Built with Next.js 15, TypeScript, and Tailwind CSS
- **SEO Friendly**: Automatic sitemap and RSS feed generation
- **Fast**: Optimized for performance with lazy loading and image optimization

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Run the development server with `npm run dev`

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Contentlayer2
- MDX
